From Idea to Device in 3 Steps

We've removed the complexity of toolchains, drivers, and libraries.

1. Connect

Plug in your ESP32 or Pico. Block IDE uses the Web Serial API to detect your board instantly directly from Chrome or Edge.

2. Drag & Drop

Snap together logic blocks. Use the Live Simulator to test your circuit logic before you even flash the real hardware.

3. Run Real-Time

Click "Run". The Python code is injected into the board's RAM. Change a value? It updates instantly on the device.

Recommended for IoT

CoresDual Core (240 MHz)
ConnectivityWi-Fi (802.11 b/g/n) + Bluetooth 4.2 / BLE

Digital I/O32 GPIO Pins (PWM capable on all)
